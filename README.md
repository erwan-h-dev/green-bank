# Green Bank
# ==========

# Hackathon project for the Ascynconf 2023.

## Installation
1. Clone the repository and move to project folder
```bash
git clone git@github.com:erwan-h-dev/green-bank.git
cd green-bank
```

2. Install the dependencies
```bash	
npm install
```

3. Run the server
```bash
npm run dev
```
